# AI Quota Tracker

A local, client-side dashboard to track AI agent usage quotas, reset cycles, and statuses. 
Built for privacy and simplicity: all data is stored securely in your browser's local storage.

## Features
- **Zero Server Setup**: Runs entirely in the browser (HTML, CSS, Vanilla JS).
- **Dynamic Services**: Add any custom AI service (ChatGPT, Claude, Custom APIs) with customizable icons, gradients, and free/pro reset cycle days.
- **Cycle & Quota Tracking**: Visual progress bars and countdowns for when your limits will reset.
- **Day/Night Theme**: Automatically switches to a dark, sleek UI at night and a clean, light UI during the day.
- **Import/Export**: Easy JSON backup of your tracked accounts and custom services.

## Usage
Simply download the repository and open `index.html` in any modern web browser.
